# Homebrew tap for [Clyde](https://github.com/kl0sin/clyde)

```sh
brew tap kl0sin/tap
brew install --cask clyde
```

That's it. Clyde will keep itself up to date through Sparkle from then
on, so `brew upgrade --cask clyde` is optional.

The cask is updated automatically by Clyde's release workflow whenever
a new signed + notarized DMG ships.
